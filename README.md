# Online Shop Sparepart PC - Sistem Informasi Spesifikasi & Rilis

## Deskripsi Proyek
Aplikasi berbasis web ini dirancang untuk mempermudah transaksi pembelian komponen komputer serta menyediakan pusat informasi teknis mengenai spesifikasi dan tanggal rilis produk.

## Struktur Repository
- **frontend/**: Berisi antarmuka pengguna (HTML, CSS, JS).
- **backend/**: Berisi logika server dan pengolahan data (PHP).
- **database/**: Berisi skema database MySQL.

## Teknologi yang Digunakan
- **Server**: Laragon (Apache & MySQL)
- **Frontend**: Bootstrap 5
- **Backend**: PHP Native / Laravel
- **Database**: MySQL
